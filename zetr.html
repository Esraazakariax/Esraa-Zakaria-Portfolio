ZETR PROJECT

Zetr - Modern Minimal Web UI

Overview:

Zetr is a modern, minimalistic web design concept built to showcase elegant UI and smooth user experience. This project highlights my ability to combine clean layout design with engaging scroll animations and a custom cursor for a premium visual feel.

Features:

● Custom animated cursor

● Smooth scroll effects using JavaScript

● Clean, responsive layout

● Modern typography and color palette

● Designed with user interaction in mind

Technologies Used:

HTML, CSS, JavaScript

Project Goal:

This project was created to explore experimental UI techniques and improve animation handling for frontend design. It's a creative concept that demonstrates my visual direction and frontend development skills.
