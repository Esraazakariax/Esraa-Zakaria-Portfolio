ROXIE PROJECT

NightCamelz – Bold Dark Web Design

Overview:

NightCamelz is a dark-themed experimental landing page with a bold visual identity. Featuring a custom cursor, stylish transitions, and smooth scroll effects, this project demonstrates my creative direction and ability to design immersive UI experiences with a unique character.

Features:

● Custom animated cursor

● Smooth, JavaScript-powered scroll animation

● Unique color palette and brand vibe

● Clean layout and responsive design

● Focus on UI mood and aesthetic consistency

Technologies Used:

HTML, CSS, JavaScript

Project Goal:

NightCamelz was built to experiment with strong branding in web design. It allowed me to explore bold visuals, interactive motion, and the emotional feel of a webpage. It reflects how I combine technical skills with storytelling through design.
